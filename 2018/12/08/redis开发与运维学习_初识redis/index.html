redis开发与运维学习（第1章 初识redis)

发表于 2018-12-08 更新于 2020-03-07 分类于 redis 阅读次数：
本文字数： 402 阅读时长 ≈ 1 分钟

1. 安装redis

  1. $ wget http://download.redis.io/releases/redis-3.2.2.tar.gz
  2. $ tar xzf redis-3.2.3.tar.gz
  3. $ cd redis-3.2.3
  4. $ make
  5. $ make install(将Redis的相关运行文件写到系统文件中，这样就可以在任意目录下执行redis的命令，需要root权限)
  6. $ redis-cli -v 查看版本

2. 启动和停止redis

  1. redis-server (默认配置启动)
  2. redis-server –key1 value1 –key2 value2 (加上要修改的配置名和值（可以多对），未设置的按照默认配置)
  3. redis-server 配置文件redis.conf路径（配置文件启动)
  4. redis-cli shutdown nosave|save 停止redis服务(是否生成持久化文件)
